Trata provides robust RESTful APIs that follows OpenAPI standards, allowing you to seamlessly integrate our services into your applications.

## Authentication

All API requests require authentication using an API key. You can obtain your API key from the [app settings page](https://app.trata.ai/settings).

<Frame>
<img src="/images/api_key.png" alt="API Key Location" />
</Frame>

Include your API key in the request header:

```bash
Authorization: Bearer YOUR_API_KEY
```

### Example Request

```javascript
fetch('https://api.trata.ai/v1/users', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  }
})
```

## Pagination

Trata implements offset-based pagination for list apis to help you navigate through large datasets efficiently.

### Pagination Parameters

| Parameter | Type    | Description                                      |
|-----------|---------|--------------------------------------------------|
| skip      | integer | Starting position of the results (default: 0)    |
| limit     | integer | Number of items to return per page (default: 10) |

### Example Paginated Request

```javascript
// Fetch the second page with 10 items per page
fetch('https://api.trata.ai/v1/items?skip=10&limit=10', {
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  }
})
```
